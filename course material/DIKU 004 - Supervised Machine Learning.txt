Artificial Intelligence Overview

Artificial Intelligence (AI) is any machine trait similar to human intelligence.

Strong/General AI: Machines that can reason across domains (research-oriented).

Weak/Narrow AI: Systems that act in specific contexts, automating decision-making.

Most modern applications use narrow AIâ€”systems that perceive part of the world, process inputs, and respond intelligently.

Applications of AI

AI powers many processes, such as:

Automation: Assembly lines, expert systems, fraud detection, self-driving vehicles, fast trading.

Insight generation: Business analytics, data pattern recognition, sciences.

Engagement: User interfaces, chatbots, voice assistants, entertainment systems.

Definition of AI

A machine has artificial intelligence if it can interpret data, learn from it, adapt, and act toward goals autonomously.

Types of AI

Symbolic AI (GOFAI): Logic-based, rule-driven reasoning.

Machine Learning: Data-driven pattern recognition.

Supervised learning: Learn from labeled examples.

Unsupervised learning: Discover hidden patterns.

Reinforcement learning: Learn via feedback.

Genetic algorithms: Use mutation and selection.

Deep Learning: Neural-network-based pattern extraction from unstructured data.

Symbolic AI manipulates logical symbols to represent knowledge; modern AI combines this with learning systems.

Symbolic AI and Logic

World modeled using symbols (nouns) and relations (verbs/adjectives).

Logic (AND, OR, NOT) combines relations.

Knowledge base: collection of all symbols and relations (used by expert systems like Siri).

Inference: discovering new true propositions through logical reasoning.

Problems: scaling, ambiguity, and dynamic concepts.
Fuzzy logic introduced degrees of truth (values between 0 and 1) to handle uncertainty.
Applications: home appliances, subway control (e.g., Sendai Subway fuzzy logic system).

What is Learning?

Learning means behavioral change from experience.
Machines learn by building models from data.

Models are simplified representations:

Descriptive (current state)

Predictive/Retrodictive (trends over time)

Mechanistic (causal processes)

Normative (optimal strategies)

Classification

Classification divides data into categories based on known labels.
Example: predict if a customer will buy a product.
Requires:

Labeled data (training set)

Testing data (for validation)

Applications: health (disease prediction), marketing, image or text classification.

Model evaluation uses confusion matrices to measure true vs. false predictions.

Classification boundaries can be linear or complex (curved, multidimensional).

Summary

Supervised Learning: Learning from labeled data to predict or classify outcomes.

Regression: Predict continuous outcomes.

Classification: Predict categorical outcomes.

Performance: Measured by accuracy or error metrics.
